# Banco de Dados da Cafeteria

## Este repositório contém a modelagem e o script SQL utilizados para criar o banco de dados do sistema da cafeteria. 
## O banco inclui tabelas de produtos e pedidos, com relacionamentos e registros iniciais.

### Como executar

### Abra seu SGBD (MySQL, MariaDB ou outro compatível).

Rode o script localizado em:
banco_cafeteria.sql

### Tecnologias utilizadas

SQL

Git e GitHub para versionamento
